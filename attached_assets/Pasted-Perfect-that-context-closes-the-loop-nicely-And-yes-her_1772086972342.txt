Simple tooltip component pattern (fast wiring)

If you’re using a UI toolkit (Radix/shadcn, etc.), great. If not, this is a lightweight pattern:

1) Store tooltips in one file

client/src/tooltips.ts exporting the JSON object.

2) Create <HelpTip id="..."> component

reads from the dictionary by id

shows title/body + “Changes picks: …”

falls back gracefully if missing

3) Add HelpTip next to each control label

Example:

Rolling Walk-Forward <HelpTip id="benchmark.mode.rollingWalkForward" />
